Seasonal Offers

This application is used by Publishers and Advertisers in Rakuten Marketing's Affiliate network. The Advertisers post upcoming promotions that are season/holiday related to a public page. Then the Publishers go on the main site and use to promotion links to offer discounts to their consumers.

https://marcom.linkshare.com

PHP, Laravel, Angular, Javascript, HTML, CSS, Mariadb, Docker

Augmented Reality Beer Pong

AR Beer Pong is a project I built for a Rakuten Mobile AR Hackathon held in December 2017. The user places a triangle of cups onto a flat surface and drags it to the desired position. Once set, the user then swipes to throw the ball into each of the cups. This project won 2nd place overall.

ARKit, Unity, C#, Swift

Googalytics .csv Exporter

The .csv Exporter was simply a script that allowed the user to pull google analytics data for multiple projects at the same time. The data is packaged in a basic front end table and then the user can export it as a .csv file from there.

JavaScript, Node, HTML, Google Analytics API
